## Status vocabulary

Statuses use the canonical triage labels defined in
`.claude/docs/triage-labels.md` (`ready-for-agent`, `ready-for-human`,
`wontfix`). That file is the source of truth; do not invent new status strings.
A feature with no remaining eligible issues is simply skipped.
